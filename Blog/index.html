Materi

  • #include : Ini adalah perintah pra-prosesor yang memungkinkan penggunaan input/output stream dari C++.
  • using namespace std;: Ini memungkinkan penggunaan elemen dari namespace std tanpa menyebutkan std:: setiap kali. Namespace std berisi banyak fungsi dasar dari C++.
  • int a;: Mendeklarasikan variabel 'a' sebagai integer.
  • main(): Ini adalah fungsi utama di mana program dimulai.
  • a = 3;: Menginisialisasi variabel 'a' dengan nilai 3.
  • do { ... } while (a <= 8);: Ini adalah loop do-while. Pada blok do, perintah cout << a << " "; akan mencetak nilai 'a' ke layar, dan kemudian nilai 'a' akan ditambah 1 dengan pernyataan a = a + 1;. Loop ini akan terus berjalan selama nilai 'a' kurang dari atau sama dengan 8.
